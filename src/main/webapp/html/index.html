Current Release 0.2 Information

The CollectionSpace project team is pleased to announce the release of CollectionSpace 0.2, which allows users to login to the system, create a new intake record, and auto-fill intake and object identification numbers. Updates to the object, create new main page, and find/edit main page are also incorporated into this release. The 0.2 release interface only allows for text entry; dates, currency, and vocabularies will be functional at a later date.
